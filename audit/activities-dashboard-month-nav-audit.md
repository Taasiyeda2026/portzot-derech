# בדיקת רכיב ניווט חודשים בעמוד פעילויות

תאריך בדיקה: 2026-06-07

## ממצאים

- בוצע חיפוש בקוד המקור אחר עמוד פעילויות/דשבורד ורכיב ניווט חודשים באמצעות המחרוזות:
  - `ניהול פעילויות`
  - `פעילויות`
  - `activities`
  - `activity`
  - `month`
  - `חודש`
  - `dashboard`
- בקוד הנוכחי של הריפוזיטורי לא נמצא עמוד דשבורד פעילויות או קובץ UI שמרנדר טבלת פעילויות, סינונים, או רכיב ניווט חודשים.
- האזכורים היחידים ל־`פעילויות` בקוד המקור שאינו `attached_assets` שייכים לתוכן לימודי סטטי ואינם עמוד דשבורד פעילויות.
- לכן לא ניתן להחזיר את הרכיב למבנה ה־render או לתקן CSS של הרכיב בלי קובצי הדשבורד הרלוונטיים.

## מה כן עודכן

- גרסת ה־Service Worker הועלתה כדי לוודא שכל שינוי עתידי בקובצי HTML/JS/CSS ייטען דרך cache חדש.
- מנגנון ה־Service Worker הקיים כבר כולל:
  - `self.skipWaiting()` בשלב ההתקנה.
  - `self.clients.claim()` בשלב ההפעלה.
  - מחיקת caches ישנים ב־`activate`.
  - Network-first עבור HTML / JS / CSS / JSON כדי למנוע תקיעה על גרסאות ישנות.

## המשך נדרש

כדי לבצע את תיקון ה־UI עצמו יש לצרף לריפוזיטורי את קובצי עמוד הפעילויות/הדשבורד, למשל רכיבי React/Vue/HTML/CSS שמכילים את:

- שורת בחירת תצוגת שבוע/חודש.
- שורת חיפוש וסינונים.
- טבלת הפעילויות.
- state/handlers של החודש הנוכחי.

לאחר צירוף הקבצים הרלוונטיים, יש למקם את רכיב הניווט כך:

```text
חץ חודש קודם | ניהול פעילויות · שם החודש · מספר פעילויות | חץ חודש הבא
```

כאשר הוא ממורכז, נמצא מתחת לבחירת שבוע/חודש אם קיימת, ומעל שורת החיפוש והסינונים.
