# משחק קלפים סטטי למדריכה

מימוש רשמי חדש ונקי למשחק קלפים ב-`HTML/CSS/JS` בלבד (ללא ספריות חיצוניות), מותאם לפריסה ב-GitHub Pages.

## מה כולל המימוש

- בחירת מצב קבוצה: **קבוצה 1 / קבוצה 2** (משפיע על Theme).
- קלט מספר משתתפות עם ולידציה (`N >= 2`).
- אלגוריתם חלוקה לתתי-קבוצות של **2 או 3 בלבד** (עדיפות ל-3, ללא קבוצה בגודל 1).
- יצירת `N` כרטיסים:
  - צד קדמי: `משתתפת #i`
  - צד אחורי: `מידע נוסף #i`
- הצגת כרטיסים תחת כותרות תתי-קבוצה: `קבוצה A`, `קבוצה B` וכו'.
- תמיכה ב-Flip תלת-ממדי לכרטיס + הגדלה בלחיצה, עם כלל: רק כרטיס אחד פתוח בכל רגע.
- לחיצה על רקע הלוח סוגרת כרטיס פתוח.
- מצב **ללא גלילה** (`overflow: hidden`) עם `layout guard`:
  - מקטין את גודל הכרטיסים בצעדים אם יש overflow.
  - אם עדיין לא נכנס למסך: מציג הודעה `המסך קטן מדי להצגה ללא גלילה` ומסתיר את הגריד.

## איך להריץ מקומית

אפשרות 1 (מומלץ): שרת סטטי פשוט מתוך תיקיית הפרויקט.

```bash
cd /workspace/portzot-derech
python3 -m http.server 8000
```

ואז לפתוח בדפדפן:

```text
http://localhost:8000/p.nativ/game/
```

אפשרות 2: לפתוח ישירות את `index.html` (ייתכן הבדל קל בין דפדפנים בניהול assets).

## פריסה ל-GitHub Pages

1. מבצעים commit + push לענף הרצוי.
2. נכנסים ל-GitHub → Settings → Pages.
3. תחת **Build and deployment** בוחרים:
   - Source: `Deploy from a branch`
   - Branch: `main` (או הענף הרלוונטי), תיקייה `/ (root)`
4. שומרים וממתינים ל-deploy.
5. כתובת המשחק תהיה:

```text
https://<username>.github.io/<repo>/p.nativ/game/
```
